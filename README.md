# RPG-Learner
RPG-Learner est une application web pédagogique innovante,  proposant une nouvelle alternative aux professeurs mais surtout aux élèves. Cette application a pour but de dynamiser l'expérience d’apprentissage des élèves via une interface orienté jeux vidéo/ RPG. Le professeur est maître de sa classe et via “une interface” peut entièrement personnaliser le contenu proposé au élèves , ajouter ,modifier ou supprimer des sujets, des questions et des réponses à celle-ci. L’élève pourra choisir entre RAID, DUEL 1V1, CLASH Classe VS Classe. Libre de a lui de profiter comme il le souhaite de l’univers qui est à sa disposition pour progresser (Lors de raid ,duel ou clash l’élève devra répondre au question proposé par le professeur)
L’élève n’a plus qu’à profiter de l'expérience vidéoludique qui lui est offert dans un univers différent des cours habituel.
